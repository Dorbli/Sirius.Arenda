## Описание
Sirius.Arenda - API для сервиса бронирования пространств, комнат и любых других помещений. Содержит в себе три группы: rooms (создание и управление помещениями), bookings (создание и управление бронирований), auth (регистрация и авторизация пользователей).

## Установка
Для установки всех библиотек, необходимых для запуска можно провести через команду в консоли в папке проекта.

```bash
pip install -r requirements.txt
``` 

## Использование
Главный файл - `app/main.py`. Достаточно только запустить главный файл, за счёт библиотеки `uvicorn` автоматически запустится API на локалхосте (http://localhost:8000).

## Документация API
Документация API хранится в файле openapi.json. Так же этот json можно посмотреть через встроенный Swagger UI в эндпоинте `/docs` или же скачать его в эндпоинте `/openapi.json`.
